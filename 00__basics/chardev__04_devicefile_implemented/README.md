# Hello File Module

The source was tested compiled and running on 5.4.75.  


## Usage

```
$ make

$ sudo insmod ./hellofile.ko

$ sudo rmmod hellofile

$ dmesg | tail
    

```


## Notes

Demonstrates a character device and device file setup.
